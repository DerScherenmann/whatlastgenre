# whatlastgenre tags and filters file
#
# [basictags]   some basic tags, mainly to differentiate between very
#               similar but different tags and for use as splitpart
# [uppercase]   tags with length>2 which should be uppercase
# [dontsplit]   regex tags that shouldn't get split
# [splitpart]   regex parts of tags that should get split by space too
# [replaceme]   regex replacements: pattern = replacement
# [filter_<name>[_fuzzy]] sections: regular expressions filters
#   lines will get surrounded by .* if you append _fuzzy
#   reserved names: album, blacklist
#
# all lines should be lowercase
# empty lines and lines starting with # are ignored
